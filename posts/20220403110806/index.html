《C++ How to Program》 06 类的模板：array和vector

1. 介绍

本章介绍数据结构的话题

  • arrays
    • 固定尺寸大数组集合
  • vectors
    • 可增长大小的数组集合

不管是array还是vector，都是C++标准库里的类模板，使用前需要包含相对应的头文件。

2. arrays

An array is a contiguous group of memory locations that all have the same type. 数组是一段连续的内存地址，保存着相同类型的数据。

我们以一段保存了12个整数的数组为例，看一下它的本质。

  • 数组共有12个元素
  • 数组名 + 元素指标(index/subscript)
    • 首个元素的指标是 0
    • 指标必须是整数类型，或者整数类型的表达式
  • 数组自由成员函数 c.size() 来获取它的大小

注意区别 “seventh element of the array” 和 “array element 7”
数组的第7个元素，实际上指标应该是 6
array element 7 的指标就是 7
推荐直接使用c[7] 这种写法

2.1. 数组的声明和初始化

数组在内存中占有位置，在使用前需要先声明。下面的写法实际上表明了数组是一种类的模板。

编译器会在声明后，根据元素的类型和数组的大小，为数组保留一定的内存空间。

3. vectors

vector 和 array 类型，但是提供 dynamic resizing 特性。使用前需要包含<vector>头文件，而且属于namespace std。

我们讨论一个代码示例如下：

  • creating vector object
    • vector<int> integers1(7);
      • 数据类型是int，名字是integers1，大小是7
      • 所有的元素默认值都是0
    • vector<int> integers1{7}
      • 这种写法表示vector对象integers1只有一个元素，该元素为7
  • vector对象同样自带函数.size()
  • 成员函数void outputVector(const vector<int>& items;
    • 在传递vector的时候，使用了引用传递，优化性能
    • const限定
    • range-based for statement
  • 成员函数void inputVector(vector<int>&;
    • 引用传递
    • range-based for statement
    • 正因为是引用传递，所以对应的输入值直接替换掉了对应的元素
  • integers1 != integers2/ integers1 == integers2
    • vector可以互相比较
  • integers1 = integers2/ vector<int> integers3{integers1}
    • vector可以由另一个vector对象赋值或者直接初始化
    • 因为vector是动态数组，所以大小不同也可以
  • integers1[5]使用具体的元素
    • 注意C++中的数组序号从0开始
    • 需要注意的是，C++不会检查序号是否溢出，所以需要程序员在调用数组元素时候，注意不要超出数组范围
  • try语句
    • Exception handing: out-of-range subscript
    • 后面会更加详细的讨论这个问题
    • 如果 try block 中的语句运行正确，则跳过 catch block 中的语句
    • 注意写法，需要花括号界定 block
    • vector的函数 at
      • 提供了边界检查，如果序号错误，则会返回exception
    • exception parameter
      • 它的成员函数what
  • integers3.push_back(1000);
    • vector成员函数push_back把vector的最后一个元素变成1000

4. 小结

要点：

  • 按引用传递数组
  • range-based for statement 来操作数组元素
  • array及其成员函数
  • vector及其成员函数
