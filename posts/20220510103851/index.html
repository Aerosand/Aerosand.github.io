OpenFOAM 2022-05-10 一些常用linux命令

发表于2022-05-10|更新于2022-06-17|OpenFOAM
|字数总计:1.1k|阅读时长:4分钟|阅读量:|评论数:

Compile C++ code in terminal

  1. create the cpp file
    touch name.cpp
    vi name.cpp
  2. compiling
    g++ -o name name.cpp
  3. run the code
    ./name

change the host name

sudo gedit /etc/hosts
sudo gedit /etc/hostname
change the name to user name

控制台常用命令

命令 说明
ls/l 列出文件
ls -l 详细列出文件
cd ~ 进入 home 目录
cd - 进入上一次工作路径
pwd 查看当前路径
mkdir filename 当前路径新建文件夹
mkdir -p [file]/[file]/filename 当前路径新建多级文件夹
rm filename 删除任何.log 文件，删除前询问确认
rm -rf [filename] 删除 filename 的子目录
rm -i *.log 删除当前路径下的 filename
rmdir [filmename] 删除非当前路径下的非空目录
rmdir -p [filename]/[filename] 删除路径下子目录
mv filename filename_new 重命名
mv file1 file2 file3 /[filename] 将文件1/2/3移动到 [filename]
mv -i filename filename_new 修改文件名，并询问存在是否覆盖
mv * ../ 移动当前路径所有文件文件夹到上一级目录
cp -r 复制目录及目录内所有项目
cp -a 复制后信息戳保持和源文件相同
xdg-open filepath/filename 控制台打开文件
gedit filename 打开文件

more 命令

跟随参数

参数 说明
+n 从第n行开始显示
-n 定义屏幕大小为n行
+/keyword 搜寻keyword，并从该字串前两行向后开始显示
-c 从顶部清理屏幕，然后显示
-d 提示按空格继续显示，q退出显示

使用中的操作命令

命令 说明
Enter 向下多显示1行
空格/ctrl+F 向下滚动一屏
ctrl+B 返回上一屏
= 输出当前行行号
:f 输出文件名和当前行号
q 退出

其他常用指令见OpenFOAM的学习

Vim 编辑

移动光标

普通模式下，

输入 说明
j 向下
nj 向下移动n行
k 向上
h 向左
l 向右
0 到行首
^ 到行首第一个字符（如果前面有空格的话）
$ 到行尾
gg 快速到文件头
G 快速到文件尾
nG 跳转到第n行

复制、剪切、删除、粘贴

输入 说明
yy 复制一行
nyy 向下复制n行
yw 复制光标开始的第一个单词
y$ 复制光标到行尾
yfA 复制光标到第一个大写的A中间的内容
y2fA 复制光标到第二个大写的A中间的内容
输入 说明
x 向后剪切一个字符，如果是在行尾，则向前
nx 向后剪切4个字符
xp 非行尾交换两个字符
输入 说明
dd 删除一行
ndd 删除n行
dw 删除一个单词
df” 删除到出现的第一个双引号
d^ 删除到行尾

注意删除的内容也会放到剪贴板，按p即可粘贴

输入 说明
p 粘贴复制
np 粘贴复制n次

替换和撤销

输入 说明
r + <字母> 将光标所在字母替换为新字母
R 连续替换知道按下ESC
cc 替换整行，即删除游标所在行，并进入插入模式
cw 替换一个单词，即删除一个单词，并进入插入模式
u{n} 撤销一次或者n次操作
U 撤销当前行的所有修改
ctrl + r redo，撤销之前的undo

退出编辑器

输入 说明
wq 保存并退出
q！ 不保存，强制退出
e！ 放弃所有编辑恢复到上一次保存
w 保存

app 进入applications文件夹
util 进入applications/utilities文件夹
sol 进入appications/solvers文件夹
run 进入$FOAM_SER_DIR/run文件夹
src 进入src文件夹
foamCloneCase case1 case2 复制case1 得到 case2
foamListTimes -rm 仅清除算例的时间计算结果
foamCleanCase 将算例还原到初始状态
foamInfo myFoam 给出myFoam的基本信息
foamCleanTutorials 清楚算例的所有计算结果
