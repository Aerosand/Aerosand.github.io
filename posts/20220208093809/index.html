【工作流】免费微软outlook代替滴答清单

发表于2022-02-18|更新于2022-02-08|Memo
|字数总计:779|阅读时长:2分钟|阅读量:|评论数:

前言

滴答清单很优秀，但是年费139元的价格还是让不少人望而却步，但是“多端同步”，“日历视图（直接拖动安排事项）”，“功能浅层易用”等特点确实非常优秀，值得一用。

我们可以使用免费的微软outlook来实现一样的功能。

开始微软outlook使用

界面

如果没有购买office365的话，左侧依然可以使用日历和邮件，两个部分功能，这两部分的功能也非常好用

将网页pwa应用化

我们可以把网页安装成PWA应用，并设置成开机启动，使得每天早上一打开个人笔记本，就能提醒到自己查看邮件，并安排今天的待办和日程。

以edge浏览器为例，网页安装成PWA应用的入口为：

当前网页下：头像右侧三点菜单 → Apps(应用) → Install this site as an app(将此网站安装为应用)

可以将应用设置在任务栏和开始菜单内，个人强烈建议把outlook设置打开为开机启动，强行提醒自己查看重要邮件，进行一天的计划。

关于edge浏览器和outlook邮箱

Outlook邮箱

非常推荐windows系统用户申请一个outlook邮箱作为主用邮箱。

  1. 作为个人笔记本的登陆账户，云端记住电脑的系统设置；
  2. 登陆edge浏览器，云端同步记住浏览器书签、密码、历史、插件等等个人配置；
  3. onedrive云盘目前在国内也已经基本可用；
  4. 通用office365，找到好友一起6人组队购买office365，一年每人只要83元，安全放心畅行微软生产力套件，使用1t的onedrive云端存储；
  5. 微软毕竟大厂，对于普通用户而言，安全性基本可以放心。

Edge浏览器

网上仍然有很多对edge浏览器的不屑言论，但是最新版本的edge浏览器已经足够好用，朴素的推荐原因如下：

  1. 无需另外的插件，开始菜单页足够简洁，bing的每日一图也让人总有惊喜；
  1. 国内可以云端同步outlook用户的所有个人配置;
  2. bing搜索可以找到baidu找不到的学术资源；
  3. 插件丰富，如grammerly，adblocker等
  4. 浏览器自身没有弹窗等广告和流量绑架。

Outlook日历

outlook可以满足绝大多数的使用需求，并且也在不断的更新功能。

基本界面

配合win11

快捷键 win + w 打开小组件。

配合移动端

移动端也可以轻松安排待办。

不足之处

目前发现不能智能识别输入，例如：“明天买猫粮”，并不能将“买猫粮”这件事情，智能识别安排到明天。

结语

outlook和微软生产力套件的联动本文没有介绍，总的来说，如果不使用滴答清单的话，作为windows系统的用户，outlook是比较好的选择，作为微软生产力套件的一部分，功能强大且齐全，值得信赖和使用。
