MIT8.01 经典力学Lec13简谐振动中的势能

发表于2022-05-09|更新于2022-05-09|物理
|字数总计:503|阅读时长:2分钟|评论数:

对于一个振子来说，总势能等于动能和弹性势能之和。

12kxmax2=12mx˙2+12kx2\frac{1}{2}kx_{max}^2=\frac{1}{2}m\dot x^2+\frac{1}{2}kx^2 21​kxmax2​=21​mx˙2+21​kx2

处理上述方程，有

x˙2+kmx2−kmxmax2=0\dot x^2 + \frac{k}{m}x^2-\frac{k}{m}x_{max}^2=0 x˙2+mk​x2−mk​xmax2​=0

两边同时对 x 取 t 的一阶导数，有

x¨+kmx=0\ddot x + \frac{k}{m}x = 0 x¨+mk​x=0

上面这个公式就是 Simple Harmonic Oscillation 简谐振动方程。

方程的解是：

x=xmaxcos(ωt+ϕ)x = x_{max}cos(\omega t+ \phi) x=xmax​cos(ωt+ϕ)

其中，

ω=km\omega = \sqrt{\frac{k}{m}} ω=mk​​

T=2πωT = \frac{2\pi}{\omega} T=ω2π​

对于这样的完美半圆弧面上的运动来说，物体运动到某一点，有势能UUU

U=mgR(1−cosθ)U=mgR(1-cos\theta) U=mgR(1−cosθ)

当 θ=0\theta =0θ=0，U=0U = 0U=0。θ=π/2\theta = \pi / 2θ=π/2，U=mgRU = mgRU=mgR。

dSR=dθ\frac{dS}{R} = d\theta RdS​=dθ

我们又有

dSdt=v\frac{dS}{dt} = v dtdS​=v

两边同时取对 t 的一阶导数，有

v(θ)=Rdθdt=Rθ˙v(\theta)=R\frac{d\theta}{dt} = R\dot \theta v(θ)=Rdtdθ​=Rθ˙

对于任意角度的运动有总机械能等于动能加上势能 ME=KE+PEME = KE + PEME=KE+PE

ME=12m(Rθ˙)2+mgR(1−cosθ)ME = \frac{1}{2}m(R\dot \theta)^2+mgR(1-cos\theta) ME=21​m(Rθ˙)2+mgR(1−cosθ)

当运动的角度很小时，Small angle approximation cosθ=1−θ22cos\theta = 1-\frac{\theta^2}{2}cosθ=1−2θ2​

ME=12R2θ˙2+mgRθ22ME = \frac{1}{2}R^2\dot \theta^2 + mgR\frac{\theta^2}{2} ME=21​R2θ˙2+mgR2θ2​

两边同时取 ddt\frac{d}{dt}dtd​

0=mR2θ˙θ¨+mgRθθ˙0=mR^2\dot \theta \ddot \theta + mgR\theta \dot \theta 0=mR2θ˙θ¨+mgRθθ˙

化简之后

θ¨+gRθ=0\ddot \theta + \frac{g}{R}\theta = 0 θ¨+Rg​θ=0

这个公式也是简谐运动方程 (SHO)

方程的解是，

θ=θmaxcos(ωt+ϕ)\theta = \theta_{max}cos(\omega t + \phi) θ=θmax​cos(ωt+ϕ)

ω=gR\omega = \sqrt{\frac{g}{R}} ω=Rg​​

T=2πRgT=2\pi \sqrt{\frac{R}{g}} T=2πgR​​

这个运动方程和单摆的运动方程是一样的。
